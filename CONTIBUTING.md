Contributing to Simple Interest Calculator
Thank you for your interest in improving this project! All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

How to Contribute
Bug Reports: If you find a bug, please open an issue.

Bug Fixes: Submit a Pull Request with a clear description of the fix.

Documentation: Improvements to the README or code comments are highly appreciated.

Enhancements: Have an idea for a new feature? Open an issue to discuss it!
